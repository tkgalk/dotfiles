# dotfiles
Personal configuration files of tkg.

# Usage
## Dependencies
```zsh
brew install stow
brew install git
```

## Symlinking
```
git clone git@github.com:tkgalk/dotfiles.git dotfiles
cd dotfiles
stow .
```
